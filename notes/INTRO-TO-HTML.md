## Terminal

### navigate
```bash
pwd
cd ~
cd my-folder
ls -la
```

### create and destroy
```bash
mkdir my-folder
touch my-file
rm -rf my-folder
```

### open programs
```bash
open .
code .
```

## VSCode 

`code .` in the terminal should open your current folder in VSCode

If that doesn't work, don't worry. Just open VSCode and locate the folder in the toolbar (`file > open`) to get the same effect.
